# 🚀 Deploy Grátis no Vercel (5 minutos)

## O que é Vercel?
Um servidor que roda seu código gratuitamente. Alfred fica online 24/7!

---

## 🎯 3 Passos Simples

### Passo 1: Entrar no Vercel

1. Abra: https://vercel.com
2. Clique em "Sign Up"
3. Clique em "Continue with GitHub"
4. Autorize (clique nos botões)

### Passo 2: Importar Repositório

1. Após fazer login, clique em "New Project"
2. Clique em "Import Git Repository"
3. Cole a URL do seu repositório:
```
https://github.com/Audenscompany/detranquest
```
4. Clique em "Import"

### Passo 3: Deploy

1. Clique em "Deploy"
2. Espera uns 2-3 minutos
3. Pronto! ✅

Você verá:
```
✅ Deployment successful!
```

E um link como:
```
https://seu-projeto-name.vercel.app
```

---

## 🎉 Usar no GitHub Pages

Agora o Alfred funciona:

**Opção 1: URL do Vercel**
```
https://seu-projeto-name.vercel.app/alfred.html
```

**Opção 2: GitHub Pages**
```
https://audenscompany.github.io/detranquest/alfred.html
```

Ambas vão funcionar! ✅
