# 🚀 COMECE AQUI - Alfred OS v9

## ⚡ ULTRA SIMPLES (5 minutos)

### Passo 1️⃣
Instale Node.js:
- Vá para: https://nodejs.org
- Clique no botão verde LTS
- Instale (clique next, next, next)
- Reinicie o computador

### Passo 2️⃣
Abra um terminal nesta pasta e execute:

```bash
npm run start-alfred
```

**Pronto!** ✅

O Alfred vai abrir automaticamente em seu navegador.

---

## Se não funcionar...

Se receber um erro, execute isto:

```bash
cd backend
npm install
npm start
```

Depois abra no navegador:
```
http://localhost:3000/alfred.html
```

---

## Usando o Alfred

1. Cole sua API Key (pegue em console.anthropic.com)
2. Digite uma mensagem
3. Clique Enviar
4. Funciona! ✅
